* 必要署名数は**オーナー数の半数＋1** とする。
* **オーナー数に上限を設定**する。あまりに資格付与すると、複数ウォレットを用いた人格の重複、複数名による結託、レビュー不足による監視効果の低下が懸念されるため。
* TXJPホルダー上位の希望者が満たないときは、ロールで代替するか総数を減らす。
* ファンド運用では緊急出動があるため、別途特別なロール、基準値を検討する。
